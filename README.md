# founder-kit

A command-line operating rhythm for founders and business owners.

`founder-kit` starts with a small terminal animation featuring Scout, a signal-bot mascot for the release, then prints a daily loop that keeps the work grounded in customer signal, constraints, revenue-facing moves, and cadence.

## Usage

```sh
npx founder-kit
```

After global install:

```sh
npm install -g founder-kit
founder
```

## Commands

```sh
founder
founder-kit
founder --demo
founder --json
founder --no-animation
```

## Development

```sh
npm test
npm run demo
```
